Estructura básica de una página web

        <html>

            <head>
                La etiqueta head contendrá parámetros e instrucciones únicamente para el navegador. No serán visibles para el usuario.
            </head>

            <body>
                Body contiene todo lo que vamos a mostrar al usuario, texto, imágenes, videos, ...
            </body>

        </html>

<html></html>

Este elemento engloba todo el contenido de la página y en ocasiones es conocido como el elemento raiz.

Puede contener atributos como por ejemplo la lengua en la que está escrita la página.

<head></head>

Este elemento actúa como contenedor para todos los parámetros que quieres incluir en el documento html que no serán visibles a los visitantes de la página. Incluye etiquetas para declarar las palabras clave y la descripción de la página que se mostrará en los resultados de búsqueda, el título de la misma, los enlaces a las hojas de estilo, la codificación de caracteres, ...

<body></body>

Body encierra todo el contenido que quieres mostrar a los usuarios cuando visitan tu página, ya sea texto, imágenes, videos, juegos, ...

Todo ello veremos que se estructurará con elementos como las etiquetas que veremos más adelante.
