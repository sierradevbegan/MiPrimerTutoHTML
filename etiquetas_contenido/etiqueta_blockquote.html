Etiqueta <blockquote>

¿Qué es?

La etiqueta blockquote se utiliza para indicar que el texto pertenece a otro sitio externo del cual se cita la fuente.

Es una etiqueta de bloque y contiene atributos comunes y de eventos. Tiene un atributo específico cite (no obligatorio) de tipo URL para indicar el origen de la cita. Conviene utilizarlo dentro de un párrafo aunque html5 lo permite.

Conviene entrecomillar el contenido y dejará un espacio a la izquierda.

¿Cómo se usa?

<blockquote>"Soy una cita externa"</blockquote>
