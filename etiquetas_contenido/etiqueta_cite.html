Etiqueta <cite>

¿Qué es?

Cite es un identificador de HTML que se puede usar tanto como etiqueta y como atributo. Es una etiqueta de línea y se renderiza en cursiva. Contiene atributos comunes y de evento.

Como etiqueta se usa para referenciar un libro, película canción,...

¿Cómo se usa?

De pequeño leí <cite>Harry Potter</cite> y está muy bien.
