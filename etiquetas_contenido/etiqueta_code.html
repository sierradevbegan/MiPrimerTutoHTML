Etiqueta <code>

¿Qué es?

Se utiliza para indicar al navegador que el contenido de dicha etiqueta es un bloque de código fuente de algún lenguaje de programación o de marcas. Es una etiqueta de linea y se renderiza con la fuente monospace predeterminada. Contiene atributos comunes y de eventos.

¿Cómo se usa?

Con esta sentencia <code> print("Hola mundo") </code> python imprime por pantalla.
